For the master pico, it will contain files:  
- master code (joystick) to send info, libraries vector3d, imu.py

For the slave pico, it will contain files: 
- slave code (car) to receive info, libaries hcsr04, imu, lcd_api, pico_i1c_lcd, vector3d, LCD display. 
Both each pico will have their main.py files.
